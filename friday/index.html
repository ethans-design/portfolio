what it is

Most productivity tools optimize for output — tasks completed, time tracked, streaks kept. Friday optimizes for what actually matters.

You set your current goals. Friday holds them in context for every single conversation. Check your email and it tells you which messages are relevant to your internship search — not just what arrived. Add a task and it sorts by goal priority, not the order you thought of it.

The idea came from noticing that the most sophisticated AI tools still treat each conversation as isolated. Friday doesn't forget what you're working toward.

how it works

the interesting parts

01

shared mic queue

Windows won't open the same audio device twice. The first version ran two streams — one for Vosk (wake word, 16kHz) and one for OpenAI (24kHz) — and they silently conflicted.

The fix: one sd.RawInputStream feeds a thread-safe queue. The realtime loop reads from the queue and resamples on the fly using Python's stdlib audioop.ratecv. No extra dependencies.

02

always-on websocket

Connecting to OpenAI's Realtime API takes 2–3 seconds. Opening and closing a connection on every wake word makes the assistant feel sluggish.

Friday keeps the WebSocket open permanently. A _conv_active flag controls whether audio flows — flipped instantly by wake word or hotkey. The conversation starts the moment "Friday" is recognized.

03

tool executor

Long-running tools (email, calendar, smart home API calls) would block the async event loop if called directly. A dedicated ThreadPoolExecutor runs them off-loop so voice stays responsive.

A tool_running flag suppresses the silence timer during execution — so the session doesn't time out while Friday is fetching your calendar.

04

https in one command

Mobile browsers require HTTPS for microphone access. Instead of certificates or a cloud server, Friday auto-starts a Cloudflare quick tunnel on launch.

The tunnel URL is parsed from cloudflared stdout and printed to the terminal within seconds. Open it on your phone — mic button unlocks.

  • Spotify — play, skip, volume

  • Web search via DuckDuckGo

  • Goals system — persistent, priority-ranked
